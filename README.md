# Customer Behavior Analysis

## Objective
Analyze customer transactions to identify customer segments, purchase patterns, and churn risks using RFM analysis.

## Dataset
Customer Behavior Analysis dataset from Kaggle.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

## Project Workflow
- Data Cleaning
- Feature Engineering
- RFM Analysis
- Customer Segmentation
- Data Visualization

## Key Findings
- Identified customer segments using RFM analysis.
- Analyzed purchase patterns.
- Identified customers at risk of churn.

## Recommendations
1. Reward loyal customers.
2. Re-engage at-risk customers.
3. Personalize product recommendations.
4. Focus on high-performing product categories.
5. Reduce customer churn through targeted campaigns.

## Conclusion
This project provides insights into customer behavior and helps improve customer engagement and retention.
